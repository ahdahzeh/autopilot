A — Tabbed Mission Control

Today / Pipeline / Insights as tabs. Vitals strip always visible. Today tab leads with an action queue — cards with Apply, Follow Up, and Dismiss buttons. Charts live on a separate Insights tab. Full table lives on Pipeline tab.

Pros

  • Action-first — opens to "do this next"
  • Clean separation of concerns
  • Family can jump in and immediately act
  • Charts accessible but not in the way

Cons

  • Extra click to see insights or full pipeline

B — Split View

Left 60%: Action feed with the same Apply/Follow Up/Dismiss cards. Right 40%: Always-visible context panel with stats, mini funnel, and status pills. No tabs — everything on one page. Full table via "View All" at the bottom.

Pros

  • Actions + context visible simultaneously
  • No tab switching needed
  • Information-dense for power users

Cons

  • More visual noise — both panels compete
  • Harder on mobile — columns collapse
  • Charts still need a separate view or get crammed
